**Swiggy Sales & Performance Analysis (Excel Dashboard Project)**

**Project Description:**
This project is a comprehensive data analysis and dashboarding solution built in Microsoft Excel, using food delivery data inspired by Swiggy.
The goal of this project is to analyze sales performance, customer behavior, ratings, and order trends, and present actionable insights through an interactive Excel dashboard.

**Project Objectives:**

* Analyze total sales and order performance
* Identify top cities, restaurants, and food categories
* Understand customer ratings and feedback trends
* Track orders by day, week, and quarter
* Compare Veg vs Non-Veg food performance
* Build an interactive business-ready Excel dashboard

**Key Performance Indicators (KPIs):**

* Total Sales (INR)
* Total Orders
* Average Order Value (AOV)
* Average Customer Rating
* Total Rating Count

**Dataset Information:**
The dataset contains the following columns:

* State
* City
* Order Date
* Day Name
* Week Number
* Quarter
* Restaurant Name
* Restaurant Location
* Food Category
* Dish Name
* Food Type (Veg / Non-Veg)
* Price (INR)
* Customer Rating
* Rating Count

**Project Structure:**
The Excel file consists of three main sheets:

1. Swiggy Data
* Raw transactional dataset
* Cleaned and formatted for analysis

2. Analysis
* Pivot tables and KPI calculations
* City-wise, category-wise, and restaurant-wise analysis
* Sales, orders, and ratings aggregation

3. Dashboard
* Interactive Excel dashboard
* Slicers for:
City
Restaurant
Food Category
Food Type
Quarter
Visual insights including:
Sales trends
Top-performing categories
Ratings distribution
Order volume patterns
